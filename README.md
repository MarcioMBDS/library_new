# 📚 Sistema de Biblioteca em C

Projeto simples desenvolvido em linguagem C com foco em aprendizado.

## 🚀 Funcionalidades
- Cadastro de livros
- Listagem de livros

## 🛠️ Tecnologias
- Linguagem C
- Structs
- Manipulação de strings

## 📌 Objetivo
Praticar conceitos fundamentais de programação em C, como:
- Structs
- Arrays
- Entrada e saída de dados
