# 📚 Sistema de Biblioteca em C

Projeto desenvolvido em linguagem C com foco em aprendizado de lógica de programação, estruturas de dados e manipulação de memória.

---

## 🚀 Funcionalidades

- 📖 Cadastro de livros
- 📋 Listagem de livros
- 🔄 Controle de disponibilidade (livro disponível ou emprestado)
- 👤 Registro de empréstimos
- 📑 Listagem de empréstimos

---

## 🛠️ Tecnologias e Conceitos

- Linguagem C
- Structs
- Arrays
- Alocação dinâmica (`malloc` e `calloc`)
- Manipulação de strings (`fgets`, `strcspn`)
- Controle de fluxo (`switch`, `while`)
